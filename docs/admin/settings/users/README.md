# Gestión de Usuarios

## Funcionalidades principales

- Listar usuarios
- Crear nuevo usuario
- Editar usuario
- Eliminar usuario
- Asignar roles y permisos

## Ejemplo de uso

1. Accede al módulo de Usuarios.
2. Haz clic en "Nuevo Usuario".
3. Completa los datos y asigna roles.
