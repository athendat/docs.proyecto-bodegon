# Gestión de Adjuntos

## Funcionalidades principales

- Listar archivos adjuntos
- Subir nuevo archivo
- Descargar adjuntos
- Eliminar archivos

## Ejemplo de uso

1. Accede al módulo de Adjuntos.
2. Haz clic en "Subir Archivo" para agregar un nuevo adjunto.
3. Selecciona el archivo y confirma la carga.
